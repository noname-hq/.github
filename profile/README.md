## How It Works

```
Request Assessment
       ↓
AI Agents + Human Researchers deploy in parallel
       ↓
Findings streamed in real time
       ↓
Full report generated automatically
       ↓
Retest & close the loop
```

Human researchers handle the nuanced, creative attack surface. AI agents handle the breadth — continuous scanning, pattern recognition, and report generation — so your team spends time on what matters.
